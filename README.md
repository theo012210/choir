# Choir Learning Plan

A web application for managing choir rehearsals, lesson plans, and checklists.

## Features

- **Role-based Views**: View plans as a Teacher, Leader, Part Leader, or Member.
- **Lesson Plans**: Track "What We Did" and "Coming Plans".
- **Interactive Dashboard**: Add new plans and filter by visibility.
- **Responsive Design**: Built with React and Tailwind CSS.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Open your browser at `http://localhost:5173` (or the port shown in the terminal).

## Tech Stack

- React
- Vite
- Tailwind CSS

## Deployment

This project is compatible with Cloudflare Pages.
Build the project using:
```bash
npm run build
```
The output will be in the `dist` directory.
